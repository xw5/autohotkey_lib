# 我的AutoHotKey脚本

### **简介**
AutoHotKey 简称 ahk，是一款 windows 平台下的热键脚本语言程序。能够通过自定义脚本实现一些重复性强没技术含量的自动化操作，而且占用资源极小，仅 3M 大小。
